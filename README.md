# Syntax SMP Website

Oficiální web serveru SyntaxSMP - CZ/SK Minecraft Survival Server

## Popis

Syntax SMP je moderní webová stránka pro Minecraft survival server. Obsahuje informace o serveru, komunitě a možnosti připojení.

## Funkce

- 🎮 Responsivní design
- 🎨 Moderní UI s červeným motivem
- 📱 Optimalizováno pro mobilní zařízení
- ⚡ Rychlé načítání
- 🔗 Odkazy na sociální sítě

## Sekce

- **Domů** - Hlavní stránka s informacemi o serveru
- **O Serveru** - Podrobné informace o Syntax SMP
- **Připojení** - Jak se připojit a hrát
- **Nábory** - Informace o přijímání nových hráčů
- **Pravidla** - Pravidla serveru
- **Store** - Obchod se speciálními prvky

## Technologie

- HTML5
- CSS3
- JavaScript (vanilla)
- Responsive Design

## Jak spustit

1. Klonuj repozitář
2. Otevři `index.html` v prohlížeči
3. Nebo nasaď na web hosting
